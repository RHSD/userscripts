# Youtube PIP
This userscript puts the YouTube video player into a small resizeable window as you scroll, similar to PIP mode.

## Usage  
- Click on the script.user.js file then Raw button to download it.
- Install Tampermonkey browser extension if you don't have it.  
- Click on Tampermonkey extension, then Dashboard.
- Drag and drop .js file  
- Make sure the script is enabled
